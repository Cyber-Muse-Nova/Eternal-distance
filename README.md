# Eternal-distance

Show how close you are to Elon Musk — physically and emotionally.

## How to Use

Visit the live site: `https://<your-username>.github.io/Eternal-distance/`

Click **Start**, allow location access, and watch the distance update in real time.

## Enable GitHub Pages

1. Go to your repo **Settings** → **Pages**
2. Under **Source**, select **Deploy from a branch**
3. Choose **main** branch, root `/`, and click **Save**
4. The site will be live at `https://<your-username>.github.io/Eternal-distance/`

## Features

- Real-time physical distance to Elon's known locations (Starbase TX, Fremont Factory CA, Austin HQ TX)
- Emotional distance counter in Astronomical Units (AU)
- Rotating savage log messages
- Mobile-friendly responsive design
